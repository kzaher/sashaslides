Screenshot all slides from a Google Slides presentation via Chrome DevTools Protocol. Execute the script at `.claude/skills/screenshot-slides.sh`.

Arguments: <presentation URL> [output directory]

Example: /screenshot-slides https://docs.google.com/presentation/d/ABC123/edit presentations/1/screenshots
