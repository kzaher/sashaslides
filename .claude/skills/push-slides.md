Push markdown slides to a Google Slides presentation. Execute the script at `.claude/skills/push-slides.sh`.

Arguments: <slides.md path> <presentation URL>

Example: /push-slides presentations/1/slides.md https://docs.google.com/presentation/d/ABC123/edit
