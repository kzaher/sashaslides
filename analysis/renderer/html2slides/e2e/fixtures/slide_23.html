Complex Borders

Six border variations testing converter edge cases
1. Thick All Sides
4px solid blue border on all four sides. Uniform width and color — the simplest multi-side case.
2. Left Accent Only
6px solid red on left only. Common "accent border" pattern. No border on other three sides.
3. Dashed Border
3px dashed teal border. Tests non-solid border-style rendering. Many converters drop dash patterns.
4. Double Border
6px double gold border. The "double" style draws two parallel lines. Rarely supported in slide formats.
5. Different Colors Per Side
Each side is a different color: red top, green right, blue bottom, gold left. Tests per-side color extraction.
6. Border + Radius + Shadow
Purple border combined with border-radius: 16px and a box-shadow. Tests all three properties together.
