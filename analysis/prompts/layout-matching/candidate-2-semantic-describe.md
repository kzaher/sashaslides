# Candidate 2: Semantic Narrative — Describe

Approach: Describe the layout as a natural-language narrative focused on visual hierarchy and reading flow.

## Instructions

Write a 2-4 sentence description of the slide's layout, answering:
1. What is the primary visual element and where is it?
2. What supporting elements exist and how do they relate spatially to the primary?
3. What is the reading flow (top-down, left-to-right, center-out, Z-pattern)?
4. How much breathing room / whitespace does the slide have?

## Output format

```
TYPE: <layout type>
DESCRIPTION: <2-4 sentences describing spatial arrangement and visual hierarchy, no content words>
ELEMENTS: <comma-separated list>
FLOW: <top-down | left-right | center-out | z-pattern | f-pattern | radial>
```

Focus on relationships between elements ("large heading spans full width above a 2-column layout of bullet points and a chart") not on content.
