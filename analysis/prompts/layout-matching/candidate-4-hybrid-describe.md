# Candidate 4: Hybrid Anchor — Describe

Approach: Lead with an explicit layout category and distinctive element, then add a semantic narrative for spatial detail.

## Instructions

First classify the slide into one of these layout categories based on the DOMINANT structural feature:

- **title-only**: slide has only a title (and maybe subtitle), no body content
- **title-bullets**: title + bullet point list (3+ items visible)
- **title-body**: title + paragraph text or short content (not a list)
- **two-column**: content split into left/right halves
- **image-dominant**: a photograph or illustration covers >40% of the slide area
- **data-table**: a structured grid/table with rows and columns of data
- **chart**: a bar chart, line chart, pie chart, or other data visualization
- **section-divider**: minimal text (1-3 words), often centered, used to introduce a new section
- **closing**: final slide — contact info, thank you, call to action, or summary
- **icon-grid**: multiple small icons or illustrations arranged in a grid
- **timeline**: sequential items arranged along a time axis
- **comparison**: side-by-side comparison of 2+ items
- **quote**: a large quotation as the primary element
- **mixed**: none of the above fits cleanly

Then describe the spatial layout in 1-2 sentences, focusing on what makes this slide's arrangement distinctive.

## Output format

```
CATEGORY: <one category from the list above>
DISTINCTIVE: <the single most structurally distinctive element, e.g., "4-row data table center", "pie chart right half", "full-bleed photo with overlay text">
SPATIAL: <1-2 sentences: where is content placed, what's the visual weight distribution, how does the eye flow>
ELEMENTS: <comma-separated list of element types present>
```

Be specific about the DISTINCTIVE element — it's the primary anchor for matching. "bullet list" is too vague; "5-item bullet list with icons, left-aligned under heading" is good.
